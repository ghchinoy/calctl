# Backlog

This file contains ideas for future enhancements to `calctl`.

## CLI Enhancements

*   `calctl create`: Create a new event.
*   `calctl update [event-id]`: Update an existing event.
*   `calctl delete [event-id]`: Delete an event.
*   `calctl day`: List events for the current day.
*   `calctl month`: List events for the current month.
*   Flag to specify a different calendar ID (other than "primary").

## MCP Enhancements

*   `create_event` tool with more parameters (attendees, description, reminders, etc.).
*   `update_event` tool.
*   `delete_event` tool.

## General

*   Add support for creating and managing reminders.
*   More robust error handling and user feedback.
